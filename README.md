# 7TV Emote Search (Raycast Extension)

Search and copy 7TV emotes directly from Raycast.

## 🚀 Getting Started

1.  **Clone the repo**
2.  **Install dependencies**: `npm install`
3.  **Run in development**: `npm run dev`

## 🕹️ Commands

### Search Emotes
Find any 7TV emote by name, preview it, and copy its CDN URL or view it on the 7TV website.
